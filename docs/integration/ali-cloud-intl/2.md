在创建用户时，用户邮箱「@」符号之前的部分需要和阿里云中的用户登录名称保持一致。

进入**{{$localeConfig.brandName}} 控制台** &gt; **用户管理** > **用户列表**，点击「**新建**」按钮，填入相关信息，**注意**邮箱「@」符号之前要填入阿里云中的用户登录名称。点击「保存」。
