使用 [UsersManagementClient](/reference/sdk-for-csharp/management/UsersManagementClient.md) 的 `create` 方法创建用户：

```csharp
var email = "test@example.com";
var password = "123456";
var user = managementClient.Users.Create(new CreateUserInput()
{
  Email = email,
  Password = password,
})
```
